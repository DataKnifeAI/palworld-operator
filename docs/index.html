How it works

Three steps from CRD to a running world.

  1. Install the operator

    kubectl apply -k config/default — CRDs, RBAC, and controller land in your cluster.

  2. Declare a PalworldServer

    Apply a sample CR. The reconciler builds Deployment, PVC, ConfigMap, Secret, and Envoy Gateway routes.

  3. Join the world

    Players connect through your gateway address on the game port. Saves land on the PVC.

Connect

Ports the operator wires for a dedicated world.

  • Game 8211 / UDP
  • Steam query 27015 / UDP
  • RCON 25575 / TCP
  • REST API 8212 / TCP

REST and RCON are powerful — keep them off the public internet unless you mean it.
